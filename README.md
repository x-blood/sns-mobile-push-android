## What is this?
FCMを利用したAndroidのPush通知サンプルアプリです。

## What is this purpose?
AWS SNSからデバイスへのプッシュ通知実行確認の為に利用します。

## Operation confirmed version
- Pie(9.0)

## How to check device token
USBデバックで出力されるログで確認することができます。
